# Mission

## What is Rovo?

Rovo is a bodyweight fitness PWA — a no-equipment workout companion that runs entirely in the browser. It guides users through timed workout sessions with audio cues, animated exercise demos, and a clean dark-mode UI.

## Target User

Solo exercisers who want structured, no-fuss workouts at home or on the go. No gym membership, no equipment, no account required. Works offline after first load.

## Core Value

**Frictionless fitness.** Open the app, pick a workout, go. Everything else — pacing, rest timers, audio cues, exercise guidance — is handled automatically.

## What Makes It Different

- No login, no tracking, no ads
- Offline-capable (PWA)
- Comic-style animated exercise demos (in progress)
- Repeat workout / "one more round" mechanic
- Minimal, fast, dark-mode-only UI

## Non-Goals

- Social features / leaderboards
- Custom workout builder (possible future idea — see IDEAS.md)
- Nutrition tracking
- Wearable integration
